# 📘 خطة بناء موقع مجمع الإمام أحمد بن حنبل
### التابع للشيخ شعبان العودة

---

## 1. نظرة عامة على المشروع

موقع تعليمي أكاديمي إسلامي يقدم:
- 🎥 **محاضرات** مضمّنة من اليوتيوب (YouTube Embed / API)
- 📝 **مقالات** علمية ودعوية
- 📚 **كتب PDF** قابلة للتحميل والقراءة

**الهوية البصرية:** اللون الأزرق ومشتقاته — تصميم عربي RTL جميل وجذاب.

**التقنيات:** HTML / CSS / JavaScript / React.js / Node.js / Express / MongoDB (MERN) / YouTube API

---

## 2. هيكل الصفحات (Pages)

| الصفحة | المسار | الوصف |
|--------|--------|-------|
| الرئيسية | `/` | Hero + أحدث المحاضرات والمقالات والكتب + نبذة عن المجمع |
| المحاضرات | `/lectures` | شبكة فيديوهات يوتيوب مع فلترة حسب السلسلة/التصنيف |
| تفاصيل محاضرة | `/lectures/:id` | مشغل الفيديو + وصف + محاضرات ذات صلة |
| المقالات | `/articles` | قائمة مقالات مع بحث وتصنيفات |
| تفاصيل مقال | `/articles/:id` | عرض المقال كاملاً |
| الكتب | `/books` | شبكة كتب PDF (غلاف + وصف + زر تحميل/قراءة) |
| عن المجمع | `/about` | نبذة عن المجمع والشيخ شعبان العودة |
| تواصل معنا | `/contact` | نموذج تواصل |
| لوحة التحكم | `/admin` | إدارة المحتوى (محمية بـ JWT) |

## 5. قاعدة البيانات (MongoDB Models)

### 🎥 Lecture
```js
{
  title: String,          // عنوان المحاضرة
  youtubeUrl: String,     // رابط اليوتيوب
  youtubeId: String,      // يُستخرج تلقائياً للـ Embed
  description: String,
  series: String,         // اسم السلسلة (مثل: شرح العقيدة)
  category: String,       // عقيدة / فقه / تفسير / حديث...
  createdAt: Date
}
```

### 📝 Article
```js
{
  title: String,
  content: String,        // نص المقال (Rich Text / Markdown)
  excerpt: String,        // مقتطف للعرض في البطاقة
  category: String,
  coverImage: String,
  createdAt: Date
}
```

### 📚 Book
```js
{
  title: String,
  author: String,
  description: String,
  coverImage: String,     // صورة الغلاف
  pdfUrl: String,         // مسار ملف الـ PDF
  pages: Number,
  category: String,
  createdAt: Date
}
```

### 👤 User (للأدمن فقط)
```js
{
  name: String,
  email: String,
  password: String,       // مشفرة بـ bcrypt
  role: { type: String, default: "admin" }
}
```

## 6. الـ API Endpoints

| Method | Endpoint | الوظيفة | محمي؟ |
|--------|----------|---------|-------|
| GET | `/api/lectures` | جلب كل المحاضرات (مع فلترة وبحث) | ❌ |
| GET | `/api/lectures/:id` | محاضرة واحدة | ❌ |
| POST | `/api/lectures` | إضافة محاضرة | ✅ JWT |
| PUT/DELETE | `/api/lectures/:id` | تعديل/حذف | ✅ JWT |
| GET | `/api/articles` | كل المقالات | ❌ |
| POST/PUT/DELETE | `/api/articles...` | إدارة المقالات | ✅ JWT |
| GET | `/api/books` | كل الكتب | ❌ |
| POST | `/api/books` | رفع كتاب (multer للـ PDF) | ✅ JWT |
| POST | `/api/auth/login` | تسجيل دخول الأدمن | ❌ |

## 7. التعامل مع اليوتيوب 🎥

**الطريقة الأبسط (موصى بها):** استخراج الـ Video ID من الرابط وتضمينه:

```jsx
function getYoutubeId(url) {
  const regex = /(?:youtube\.com\/(?:watch\?v=|embed\/)|youtu\.be\/)([\w-]{11})/;
  return url.match(regex)?.[1] || null;
}

// في المكوّن
<iframe
  src={`https://www.youtube.com/embed/${youtubeId}`}
  title={title}
  allowFullScreen
/>
```

**الصورة المصغرة تلقائياً (بدون API Key):**
```
https://img.youtube.com/vi/{VIDEO_ID}/hqdefault.jpg
```

**اختياري لاحقاً:** YouTube Data API v3 لجلب العنوان والمدة تلقائياً عند إضافة الرابط.

## 10. أفكار تصميمية لجمال الموقع ✨

1. **Hero Section** بخلفية زرقاء متدرجة مع زخرفة إسلامية هندسية خفيفة (SVG Pattern) ولمسة ذهبية.
2. **بطاقات (Cards)** بظل ناعم وحواف دائرية `border-radius: 12px` مع Hover يرفع البطاقة قليلاً.
3. **قسم "أحدث المحاضرات"** في الرئيسية بصور مصغرة من اليوتيوب.
4. **شريط تصنيفات** أفقي (عقيدة، فقه، تفسير، حديث، سيرة) بأزرار Pill زرقاء.
5. **Dark Mode اختياري** لاحقاً بدرجات الأزرق الداكن.
6. **أنيميشن خفيف** عند التمرير (Intersection Observer أو مكتبة AOS).
